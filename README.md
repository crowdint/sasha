# Sasha

Add your current Git SHA to your Rails app
response headers.

## Requirements

git? rails?

## Usage

### Rails 3 or Rails 2 with Bundler

Add it to your Gemfile, you are using bundler, right?

    gem 'sasha'

## Test it

    $ curl -I http://www.example.net | grep X-Git-SHA
